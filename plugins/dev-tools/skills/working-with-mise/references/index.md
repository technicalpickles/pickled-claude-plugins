# mise Reference Documentation

This directory contains documentation fetched from [mise.jdx.dev](https://mise.jdx.dev/).

Run `./update-docs.sh` to refresh the documentation.

## Directory Structure

### cli/

- `use-html.md` - Adding tools to config
- `doctor-html.md` - Diagnostic commands
- `activate-html.md` - Shell activation
- `which-html.md` - Path resolution
- `ls-html.md` - Listing installed tools
- `exec-html.md` - Running commands with mise context
- `trust-html.md` - Trusting config files
- `install-html.md` - Installing tools
- `outdated-html.md` - Checking for updates
- `upgrade-html.md` - Upgrading tools

### dev-tools/
Tool management concepts:
- `index-html.md` - Dev tools overview
- `shims-html.md` - Shims vs PATH activation
- `aliases-html.md` - Version aliases
- `backends.md` - Backend overview
- `backends/` - Individual backend docs (npm, cargo, pipx, etc.)

### configuration/
Configuration reference:
- `settings-html.md` - mise settings
- `environments-html.md` - Environment management

### guides/
Getting started and tutorials:
- `getting-started.md` - Initial setup guide
