# x-twitter-scraper

Plan X/Twitter data workflows with Xquik REST API routing, MCP setup, SDKs, webhooks, exports, monitoring, and confirmation-gated actions.

## What It Does

- Choose the narrowest Xquik route for tweet, profile, timeline, follower, or search data.
- Prepare an MCP setup checklist for an agent workflow.
- Plan a monitor or webhook workflow with HMAC verification.
- Draft SDK or OpenAPI integration steps for an app or dashboard.
- Add an explicit approval step before private reads, persistent work, or write actions.

## Install

```bash
/plugin install x-twitter-scraper@pickled-claude-plugins
```
